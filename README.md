# Python-file-handling
Python for file handling: search, copy, delete lines, manage contacts, and task tracking.

## 📌 Overview
This project contains simple Python exercises focused on file handling.

## ⚙️ Features
- Search for a word in a file
- Copy content from one file to another
- Delete lines containing a specific word
- Manage contacts (add & search)
- Simple task manager

## 🛠️ Technologies
- Python (file handling, basic IO)

## 🎯 Purpose
Practice and improve understanding of file operations in Python.
